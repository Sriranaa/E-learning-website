# 🎓 Upskilling - E-Learning Platform

## 📌 Project Overview

Upskilling is a static e-learning website developed using HTML, CSS, JavaScript, XML, and XSD.
The platform allows users to explore various courses, view detailed course information, and interact through forms.

---

## 🚀 Features

* 🏠 **Home Page**

  * Navigation bar
  * Banner section
  * Login form

* 📚 **Courses Page**

  * Course cards with images
  * Course description, instructor details
  * Curriculum, duration, and enroll button
  * Glow effects and hover animations

* 📖 **About Page**

  * Information about the platform
  * Mission, vision, and offerings

* 📞 **Contact Page**

  * Address, email, and phone details
  * Contact form for user queries

* 🔐 **Login Validation**

  * JavaScript-based validation
  * Checks for empty fields
  * Displays custom popup messages

* 📄 **XML & XSD**

  * XML file stores course data
  * XSD file validates XML structure

---

## 🔐 Form Validation

* Implemented using JavaScript (`script.js`)
* Validates:

  * Empty input fields
  * Basic input checks
* Displays custom popup messages instead of default alerts

---

## 📊 XML & XSD Implementation

* **XML (`courses.xml`)**

  * Stores structured data of all courses
  * Includes title, instructor, description, duration, price, rating, and curriculum

* **XSD (`courses.xsd`)**

  * Validates XML structure
  * Ensures correct data types and hierarchy
